A plugin can be ignored after the installation process. If the plugin cannot be called by laravel because of the a missing value means its active, a value of off, no, 0 is going to have this ignored

 "extra": {
     "laravel": {
         "plugin_active" : "Yes"
     }
 }
}

$auto_publish_paths

PluginRef.php : 55

If a plugin has resources it wants to publish, this holds the value of that found in the composer.json in the extra field

protected array{alias: string, relative_path: string} $auto_publish_paths

For example, this may be in the json, then the auto_publish_paths will hold the values. Both the expected values need to be in the json or this will be empty

 "extra": {
     "laravel": {

         "auto_publish_assets" : {
             "alias": "resources",
             "relative_path":"resources/dist"
         }
     }
}
